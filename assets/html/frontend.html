HTML

HTML (Linguagem de Marcação de HiperTexto) e a base do conteúdo da web, e como o esqueleto do de um site, fornecendo a estrutura essencial sobre a qual todo resto e construido como uma Linguagem de Marcação padrão para paginas da web.

O HTML usa tags para definir elementos e estruturar o conteúdo de titulos a paragrafos, links e imagens, tudo oque e visivel em uma pagina da web e normalmente estruturado usando HTML. Lembrando que HTML não é uma linguagem de programação; é uma linguagem de marcação, usada para definir a estrutura do seu conteúdo.

Elementos essenciais que compõem a estrutura básica do HTML:

  • ‹!DOCTYPE html›: Define o tipo de documento HTML e deve ser o primeiro elemento em um documento HTML.
  • ‹html›: Define o elemento raiz de um documento HTML e encapsula todo o conteúdo da página.
  • ‹head›: Contém informações sobre o documento, como metadados, scripts, estilos e o título da página.
  • ‹title›: Define o título da página, que é exibido na barra de título do navegador ou na guia da página.
  • ‹body›:Contém todo o conteúdo visível da página, como texto, imagens, vídeos, links e outros elementos HTML.
  • ‹h1›-‹h6›:Define cabeçalhos de diferentes níveis de importância, sendo ‹h1› o mais importante e ‹h6› o menos importante.
  • ‹p›: Define um paragrafo de texto
  • ‹a›: Define um hyperlink, ou link, que permite a navegação entre páginas ou dentro da mesma página.
  • ‹img›:Define uma imagem a ser exibida na página.
  • ‹ul›: Define uma lista não ordenada, que geralmente é uma lista de itens com marcadores.
  • ‹li›: Define um item de lista em uma lista
  • ‹div›: Define uma divisão ou seção genérica de uma página.
  • ‹span›: Define uma pequena seção de texto ou conteúdo inline.
  • ‹button›: Define um botão clicável.

    CSS

    Cascading Style Sheets (CSS) é uma linguagem de estilo utilizada para descrever a apresentação de um documento HTML. Com o CSS, é possível definir o estilo visual de elementos HTML, como cor, fonte, tamanho e posicionamento.

    Importância do CSS no Desenvolvimento Web

  • Separation of Concerns: O CSS permite separar a estrutura (HTML) do estilo (CSS), facilitando a manutenção e a organização do código.
  • Flexibilidade e Customização: Com o CSS, é possível personalizar completamente a aparência de uma página web, adaptando-a às necessidades específicas do projeto.
  • Consistência Visual: O uso consistente de estilos CSS ajuda a garantir uma experiência de usuário uniforme em todo o site.

    • Sintaxe Básica do CSS

    O CSS é composto por regras, que consistem em um seletor e um bloco de declaração de estilo:

  • Seletor: Indica qual elemento HTML será estilizado.
  • Propriedade: Define qual aspecto do elemento será estilizado (por exemplo, cor, tamanho, margem).
  • Valor: Especifica o valor da propriedade (por exemplo, vermelho, 16px, 10px).

    Propriedades básicas

  • Propriedades de texto: como estilizar textos, alterar tamanho, cor, fonte e estilo, por exemplo, font e text align.
  • Propriedades de tamanho: como definir tamanhos para containers, seções, textos ou qualquer elemento da tela, por exemplo, width e height.
  • Propriedades de posicionamento: como posicionar elementos e criar layouts para a tela, por exemplo, display e position.
  • Propriedades de espaçamento: como separar e dar espaço entre elementos, por exemplo, margin e padding.

    JAVASCRIPT

    JavaScript, frequentemente abreviado como JS, é uma linguagem de programação que permite a criação de interatividade em páginas web. Diferente do HTML, que é uma linguagem de marcação usada para estruturar o conteúdo de uma página, e do CSS, que é uma linguagem de estilo usada para estilizar o layout e a aparência visual de uma página, JavaScript é uma linguagem de programação que adiciona comportamento dinâmico às páginas web.

    Importância do JavaScript no Desenvolvimento Web

  • Interatividade: JavaScript permite a criação de elementos interativos, como botões, formulários, menus, efeitos de animação, entre outros
  • Manipulação do DOM: JavaScript permite manipular o DOM (Document Object Model), permitindo adicionar, remover ou modificar elementos HTML e CSS em uma página de forma dinâmica.
  • Reatividade: Com o surgimento de frameworks e bibliotecas JavaScript como React.js e Vue.js, JavaScript é usado para criar interfaces de usuário reativas e componentizadas que respondem às mudanças de estado de forma eficiente.

    • Sintaxe Básica do JavaScript

    1. Declaração de Variáveis:

    Em JavaScript, você pode declarar variáveis usando as palavras-chave 'var', 'let' ou 'const'.

  • var: Define uma variável de escopo global ou de função. Pode ser redeclarada e reatribuída.
  • let: Define uma variável de escopo de bloco. Pode ser reatribuída, mas não redeclarada no mesmo escopo.
  • const: Define uma variável de escopo de bloco com um valor que não pode ser alterado posteriormente.

    • 2. Tipos de Dados:

    JavaScript possui vários tipos de dados, incluindo números, strings, booleanos, arrays, objetos, entre outros.

  • string: As strings são sequências de caracteres, como texto, entre aspas simples (') ou duplas (").
  • booleanos: Booleanos representam valores lógicos 'true' ou 'false', indicando verdadeiro ou falso.
  • arrays: Arrays são coleções ordenadas de elementos, que podem ser de qualquer tipo de dados.
  • Objetos: Objetos são coleções de pares de chave-valor, onde cada chave é uma string única e cada valor pode ser de qualquer tipo de dados.

    • 3. Operadores:

    JavaScript suporta uma variedade de operadores, incluindo aritméticos, de atribuição, de comparação e lógicos.

    4. Estruturas de Controle de Fluxo:

    JavaScript suporta estruturas de controle de fluxo como condicionais (if e else)

  • if: A declaração if é usada para executar um bloco de código se uma condição específica for verdadeira.
  • else: A declaração else é usada para executar um bloco de código se nenhuma das condições anteriores for verdadeira.

    • 5. Funções:

    Você pode definir funções em JavaScript usando a palavra-chave function.

    Esses são apenas alguns exemplos da sintaxe básica do JavaScript. Com o tempo, você irá se familiarizar mais com esses conceitos e poderá utilizá-los para criar programas mais complexos e funcionais.
